####react版快速排序


###安装

项目地址：（`git clone`）

```shell
将代码克隆至本地
```

通过`npm`安装本地服务第三方依赖模块(需要已安装[Node.js](https://nodejs.org/))

```
npm install
```

启动服务(http://localhost:3006/html#/sort)

```
npm run dev
```

发布代码
```
webpack -p
```
